Frequently Asked Questions

Common questions about Mobile App.

The mobile app provides comprehensive patient engagement features including appointment booking with real-time doctor availability, digital access to lab reports and test results, complete medical history and patient records, prescription viewing and medication reminders, family member profile management, clinic and doctor search by specialty, push notifications for appointments and health updates, multi-language support (English, Arabic, and more), and integrated payment processing for consultations and services.
Yes, the MEDAS mobile app is available as native applications for both iOS and Android platforms. The iOS version is available on the Apple App Store for iPhone and iPad users, while the Android version is available on Google Play Store. Both versions offer the same comprehensive features with interfaces optimized according to each platform's design guidelines (iOS Human Interface Guidelines and Android Material Design).
The mobile app seamlessly integrates with MEDAS HIMS (Hospital Information Management System) and other healthcare management platforms through secure APIs. This integration enables real-time data synchronization including instant appointment confirmation and updates, immediate access to new lab results, automatic medical record updates, synchronized prescription information, real-time doctor availability, and integrated billing and payment processing. All patient data is synchronized across the hospital/clinic system and mobile app in real-time.
Absolutely. The mobile app includes family profile management that allows a primary user to add and manage health profiles for multiple family members including children, elderly parents, and dependents. Users can book appointments for any family member, view their individual medical records and lab reports, track vaccination schedules, manage prescriptions, and receive notifications for all family members' appointments and health updates from a single account.
Patient data security is our highest priority. The mobile app implements multiple layers of security including end-to-end encryption for all data transmission, secure user authentication with biometric login options (Face ID, Touch ID, fingerprint), encrypted local storage on the device, HIPAA and GCC healthcare compliance standards, automatic session timeout for inactive users, secure API communication with SSL/TLS protocols, and regular security audits and updates. All patient health information is protected according to international healthcare privacy regulations.
